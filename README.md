# 💰 PocketGuard: 스마트 미니 가계부 매뉴얼

PocketGuard는 번거로운 입력 과정을 최소화한 직관적인 가계부입니다.

## ✨ 주요 기능
- **원클릭 기록**: 금액 입력 후 카테고리 버튼만 누르면 즉시 기록됩니다.
- **데이터 시각화**: 
  - **원 그래프**: 카테고리별 지출 비중 확인
  - **막대 그래프**: 최근 7일간의 지출 추이 비교
- **자동 합계**: 오늘 하루 동안 쓴 총액을 실시간으로 계산해 보여줍니다.

## 🛠 설치 및 실행 방법
1. `PocketGuard.exe` 실행
2. `customtkinter`, `matplotlib` 라이브러리 필요 (소스 실행 시)
